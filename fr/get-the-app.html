Prévisualisez l'application Garabaly

Garabaly est actuellement en version préliminaire. Suivez ces deux étapes simples pour essayer l'application sur votre téléphone via Expo Go.

Étape 1

Téléchargez Expo Go

Expo Go est une application gratuite qui vous permet de prévisualiser des applications mobiles avant leur publication sur l'App Store ou Google Play. Téléchargez-la d'abord sur votre appareil.

App Store

Scannez pour télécharger Expo Go pour iPhone et iPad

Ou ouvrir le lien directement

Google Play

Scannez pour télécharger Expo Go pour Android

Ou ouvrir le lien directement
Étape 2

Scannez le QR Code Garabaly

Une fois Expo Go installé, scannez le QR code ci-dessous et appuyez sur « Ouvrir dans Expo Go » pour lancer l'application Garabaly.

QR Code de l'application Garabaly

Sur iPhone

Ouvrez l'application Appareil photo et pointez-la vers le QR code. Appuyez sur la notification qui apparaît, puis sélectionnez « Ouvrir dans Expo Go ».

Sur Android

Ouvrez l'application Expo Go et appuyez sur « Scanner le QR Code ». Pointez votre appareil photo vers le code et appuyez sur « Ouvrir dans Expo Go ».

Bientôt disponible

Application native bientôt sur les stores

Garabaly est actuellement en prévisualisation via Expo Go. L'application native complète pour iOS et Android sera bientôt disponible sur l'App Store et Google Play. Restez à l'écoute !
